# NEP-for-superionic-Ag<sub>2</sub>Te-and-Cu2<sub>2</sub>Se

1. This includes NEP mode for superionic Ag<sub>2</sub>Te and Cu<sub>2</sub>Se, with a valid temperature range from 500 to 900 K.  
2. A post-processing code for calculating Onsager's transport coefficient is also included.  
3. Contact me (meyixinx@ust.hk) if you have any questions.
